# 工具评分记录

## 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 匹配度 | 40% | 工具是否真的适合这个任务 |
| 易用性 | 30% | 引导流程是否顺畅 |
| 效果 | 30% | 最终产出质量 |

## 评分格式

```json
{
  "tool_id": "builtin-value-cost",
  "task": "选择AI组件库方案",
  "score": 8.5,
  "dimensions": {
    "match": 9,
    "usability": 8,
    "effect": 8
  },
  "timestamp": "2026-06-17T15:30:00+08:00",
  "notes": "适合价值判断类任务"
}
```

## 评分影响

累计评分（加权平均）影响该工具在路由阶段中的推荐排序。
