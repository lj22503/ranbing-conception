# 分析阶段路由器

## 触发条件

理解阶段完成后，或用户直接表达"帮我分析/判断/决定 XXX"。

## 路由逻辑（二维决策）

### 第一维：判断任务类型

| 任务类型 | 用户表述特征 | 对应框架 |
|---------|------------|---------|
| 价值判断 | "值不值得""要不要做""选哪个" | 价值-成本矩阵 / 四维评估 |
| 根源诊断 | "为什么""什么原因""根在哪" | 要素分析 / 第一性原理 |
| 战略选择 | "下一步怎么走""路线选哪个" | SWOT / 横纵分析 / 价值-成本 |
| 结构拆解 | "怎么分解""拆开看" | 要素分析 / 第一性原理 |
| 风险评估 | "有什么风险""最坏会怎样" | SWOT / 四维评估 |
