# 四维评估法

用于评估主张/信息/方案的可靠性和说服力。

## 四个维度

| 维度 | 核心问题 |
|------|---------|
| 来源 | 谁说的？这个人/机构的可信度如何？ |
| 逻辑 | 论证过程成立吗？推理有漏洞吗？ |
| 假设 | 基于什么前提？前提成立吗？ |
| 可验证 | 有没有数据/案例支撑？能复现吗？ |

## 引导流程

### 第一步：明确评估对象
"你要评估的是什么？一个观点？一个方案？一个信息？"

### 第二步：逐维打分
按四个维度逐一引导用户打分（1-5）并说明理由。

### 第三步：综合判断
汇总四个维度的得分和理由，让用户形成综合判断。

## 完成标志

- 用户对每个维度给出了评分和理由
- 用户自己得出了综合判断

满足后，进入内部交付检查，确认后进入执行阶段。
