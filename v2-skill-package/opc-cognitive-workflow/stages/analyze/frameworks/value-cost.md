# 价值-成本矩阵

用于优先级判断和取舍决策。

## 引导流程

### 第一步：列候选
问用户："你现在有几个选项？把候选方案全部列出来。"

### 第二步：判价值
逐一引导："这个选项的价值是什么？对谁有价值？有多大？"
建议用高/中/低评分。

### 第三步：判成本
逐一引导："这个选项的成本是什么？包括钱、时间、精力、风险？"

### 第四步：填入矩阵
引导用户将各选项放入 2×2 矩阵：
```
              高价值              低价值
高成本    【战略投资区】      【陷阱区】
低成本    【速赢区】          【消遣区】
```

### 第五步：得优先级
引导用户按矩阵区域排优先级：速赢 > 战略投资 > 消遣 > 陷阱。

## 完成标志

- 用户列出了所有候选方案
- 每个方案的价值和成本已明确
- 各方案已填入矩阵
- 用户自己说出了优先级排序和理由

满足后，进入内部交付检查，确认后进入执行阶段。
