# 要素分析法

用于结构拆解和根源诊断。核心原则：MECE（相互独立、完全穷尽）。

## 引导流程

### 第一步：定边界
"你要分析的对象的边界是什么？范围有多大？"

### 第二步：第一层拆解
"从最宏观的维度，它可以分成哪几个大的部分？"
确保 MECE。

### 第三步：逐层深挖
对每个关键部分继续拆解：
"这个部分可以进一步分解成什么？"

### 第四步：标关键
"这些要素中，哪些是关键要素？哪些只是细节？"

### 第五步：回到整体
"拆完了，现在回看整体，你发现了什么之前没注意到的？"

## 完成标志

- 拆解结构 MECE
- 关键要素已被标注
- 用户能从拆解中获得新的整体洞察

满足后，进入内部交付检查，确认后进入执行阶段。
