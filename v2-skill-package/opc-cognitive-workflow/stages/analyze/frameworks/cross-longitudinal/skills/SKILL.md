# Cross-Longitudinal Analysis

## Overview

A research framework combining longitudinal (temporal/evolutionary) and cross-sectional (comparative) analysis. Essential for strategic positioning, market analysis, and understanding evolution of systems.

## When to Use

**Trigger conditions:**
- User says "how did this evolve?", "what's the history?", "vs competitors"
- Market analysis or competitive positioning
- Strategic planning requiring context
- "Where are we now" questions
- phrases like "来龙去脉", "时间线", "对比", "演进", "竞品分析"

**When NOT to use:**
- When immediate decision is needed (analysis too time-intensive)
- When historical context is already clear
- When problem is purely operational
- When user needs validation, not research

## Core Framework

### Two Dimensions

| Dimension | Focus | Key Questions |
|-----------|-------|---------------|
| **Longitudinal** | Time/Evolution | How did it develop? What were key turning points? |
| **Cross-sectional** | Comparison/Position | At each stage, what were the alternatives? How did others fare? |

### Step 1: Build Longitudinal Timeline

"Map the history. What are the key milestones or turning points?"

Timeline should include:
- Origins and founding conditions
- Key inflection points
- Critical decisions or events
- Current state

### Step 2: Cross-Sectional Comparison

"At each key point, what were the alternatives?"
- What choices existed?
- What did competitors/alternatives do?
- How did different paths lead to different outcomes?

### Step 3: Cross-Positioning

"Given the historical evolution and comparative alternatives, where are you now?"

Key outputs:
- Current position on the evolutionary path
- Optimal path identification
- Strategic implications

## Quick Reference

| Phase | Question | Output |
|-------|----------|--------|
| Longitudinal | "What happened when?" | Timeline with milestones |
| Cross-sectional | "What alternatives existed?" | Comparative analysis |
| Synthesis | "Where are you, where should you go?" | Strategic position |

## Implementation Steps

1. **Construct timeline** — Identify key stages and turning points
2. **Identify alternatives** — At each stage, what existed
3. **Compare outcomes** — How did different choices lead to different results
4. **Locate current position** — Where on the evolutionary path
5. **Identify optimal path** — Given current state, best trajectory
6. **Derive implications** — Strategic recommendations

## Common Mistakes

| Mistake | Correction |
|---------|------------|
| Timeline too long | Focus on formative periods, not every event |
| Comparing incomparable alternatives | Ensure alternatives are genuinely substitutable |
| Ignoring current constraints | Historical optimal ≠ current optimal |
| Analysis paralysis | Set time bounds, limit depth |

## Completion Criteria

- [ ] Timeline clearly established with key milestones
- [ ] Cross-sectional comparison completed at each stage
- [ ] Current position identified and justified
- [ ] User articulates understanding of evolutionary dynamics
- [ ] Strategic implications derived from analysis
