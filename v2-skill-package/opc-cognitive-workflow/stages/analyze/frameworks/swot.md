# SWOT 分析

用于战略选择和风险评估。

## 引导流程

### 第一步：填四象限
分别引导用户填写：
- **S (Strengths)**：你的优势是什么？
- **W (Weaknesses)**：你的劣势是什么？
- **O (Opportunities)**：外部机会是什么？
- **T (Threats)**：外部威胁是什么？

### 第二步：交叉分析
- SO：如何用优势抓住机会？
- WO：如何用机会弥补劣势？
- ST：如何用优势抵御威胁？
- WT：如何减少劣势同时避开威胁？

### 第三步：综合判断
引导用户根据交叉分析得出综合结论和行动方向。

## 完成标志

- 四象限已填满
- 完成了交叉分析
- 用户得出了综合结论

满足后，进入内部交付检查，确认后进入执行阶段。
