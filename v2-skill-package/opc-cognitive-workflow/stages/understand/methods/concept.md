# 概念法

用于理解抽象概念。

## 引导流程

### 第一步：激活旧图式
问用户："你之前是怎么理解这个概念的？用你自己的话说说。"

确认用户用自己的语言描述了已有理解。

### 第二步：建立连接
引导："这个新概念和你已知的什么有联系？像什么？不像什么？"

建立新旧知识的连接点。

### 第三步：指出冲突
引导："它和你之前的理解有什么不同？什么地方让你意外？"

指出新旧认知的矛盾或差距。

### 第四步：检验迁移
引导："现在你可以用这个新理解去看什么问题？换一个场景你会怎么用？"

检验新理解是否可以迁移到其他场景。

## 完成标志

- 用户能用自己话说出"这是什么"
- 用户能说出"和我有什么关系"
- 用户能说出"和我想的不一样在哪"
- 用户能说出"我现在能拿它做什么"

满足后，进入内部交付检查，确认后进入分析阶段。
