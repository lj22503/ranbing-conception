# 情境法

用于理解现象或情境。

## 引导流程

### 第一步：画出现象轮廓
问用户："你观察到什么？描述你所看到/听到的具体现象。"

### 第二步：连到已有经验
引导："这个现象让你想起了什么？你以前见过类似的吗？"

### 第三步：指出不合常理处
引导："这里面什么让你觉得不对？什么地方不符合你的预期？"

### 第四步：预判走向
引导："如果不加干预，你觉得这事会怎么发展？为什么？"

## 完成标志

- 用户能清楚描述现象轮廓
- 用户能把现象和已有经验建立连接
- 用户能指出异常/意外点
- 用户能做出带理由的预判

满足后，进入内部交付检查，确认后进入分析阶段。
