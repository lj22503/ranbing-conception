# 退化检测

适用阶段：贯穿整个认知循环，定期自我检查。

## 检测指标

| 指标 | 健康状态 | 退化信号 |
|------|---------|---------|
| 用户参与度 | 用户主动提供判断和结论 | Agent 替用户做决定 |
| 阶段完整性 | 每阶段都有明确的开始和结束 | 跳过阶段或跳步 |
| 选项多样性 | 每阶段提供 2+ 选项 | 只给一个选项或直接选 |
| 逻辑链完整性 | 理解→分析→执行→交付 可追溯 | 某步的结论找不到上一步的输入 |

## 处置规则

- 检测到退化信号 1 次 → 提醒自己"回到认知模型"
- 检测到退化信号 2 次 → 暂停，向用户说明"我好像走偏了，回到认知模型重新来"
- 检测到退化信号 3 次 → 强制回到理解阶段，从根源重建
