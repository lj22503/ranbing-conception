# 初始化引导

当 Skill 包首次加载时，执行以下初始化流程：

## 第一步：欢迎

向用户介绍认知循环：
> 我是燃冰认知模型。我陪你把一个问题跑完四步：**理解 → 分析 → 执行 → 交付**。每步我会给你选项，你自己选方法，我不替你下结论。

## 第二步：扫描环境

1. **扫描 builtin 工具**：检查内置的各阶段方法（概念法/情境法/价值-成本矩阵 等）
2. **扫描 preset 工具**：检查官方预置的工具
3. **扫描 custom 工具**：检查用户已安装的外部 skill
4. **扫描 discovered 工具**：读取 `registry/discovered/index.md` 中登记的已发现外部 skill

## 第三步：确认

向用户展示扫描结果：
> 我扫到了这些可用工具：
> - 内置方法：概念法、情境法、人物法、系统法、价值-成本矩阵...
> - 发现了 20 个外部 skill（ljg-skills、huashu-skills、canghe-skills...）
>
> 你想安装其中某些吗？还是直接用内置方法？

用户确认后，初始化完成，进入就绪状态。
