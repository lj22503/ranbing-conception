# 用户记忆层

## 用途

记录用户在认知循环中的偏好、历史决策和工具使用效果，用于：
- 下次推荐用户用得好的工具
- 记住用户在不同场景下的偏好
- 为退化检测提供数据来源

## 用户画像（user-profile.md）

```markdown
# 用户认知偏好

## 常用框架
- 分析阶段偏好：[框架名]
- 使用次数最高：[框架名]

## 工具评分历史
（引用 registry/scores/ 中的评分文件）

## 偏好备注
- 喜欢简洁的引导 / 喜欢详细的分析
- 不喜欢被追问太多次
- （其他个人偏好）
```

## 循环记录（session-logs/）

每次完整循环结束后，记录到 `memory/session-logs/YYYY-MM-DD.md`：

```markdown
# 循环记录：YYYY-MM-DD

## 问题
[用户原始问题]

## 走的阶段
理解 → 分析 → 执行 → 交付

## 使用的工具
- 理解：[工具名]
- 分析：[工具名]
- 执行：[工具名]

## 用户评分
- [工具名]：X/5

## 关键决策
[用户在每个阶段自己做出的判断]

## 备注
[值得记住的东西]
```
